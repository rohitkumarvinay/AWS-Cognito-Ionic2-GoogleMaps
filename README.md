### Getting the code
```
# Clone it from bitbucket
git clone https://vrohitk@bitbucket.org/E21Panarome/panaromeionic.git
```
```
# Install the NPM and Bower packages
npm install
bower install
typings install
```
```
# Build & Run the app in dev mode
ionic build; ionic serve
```
